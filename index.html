Test type

Display speed as

Nozzle velocity (mm/s) is what Orca's PA Pattern dialog actually asks for, so it's the default. Volumetric rate (mm³/s) is the same test expressed as flow — comparable across printers and nozzles. Switching converts the values below.

Adaptive PA depends on both flow and acceleration. The point counts start from a smart default that scales with your printer's envelope — a narrow accel or flow range needs fewer points, so a low-accel/low-flow machine won't be forced into a big matrix. Both counts are yours to override, and the value lists auto-fill from them (or type your own). The test grid is every speed × every accel.
